# IRIS AGENTS AND BOTS
Bienvenidos al repositorio principal donde se alojan los agentes o bots creados en IRIS Startup Lab.

## ¿Qué se puede encontrar aquí?
- Código base de los agentes (escritura en LangChain o ADK de Google)
- Los "prompts" principales o "system prompt"
- Los íconos de los agentes
- Las tarjetas adaptativas (Para copilot studio)
- El código frontend o backend

## ¿Cuál es la estructura de este repositorio?

```
IRIS AGENTS AND BOTS/
├── chismesito_gpt/                      # Carpeta que enlaza con chismesitoGPT
├── copilot_agents_and_bots/             # Carpeta con los bots realizados en Copilot Studio (la mayoría)
│   ├── bots_chatgpt_to_copilot/         # Bots migrados de chatgpt a copilot
│   │   ├── bots/                        # Carpeta de bots
│   │   │   └── andyideas/               # Bot llamado AndyIdeas
│   │   │   └── chris_detective/         # Bot llamado Chris Detective
│   │   ├── iconos_bots/                 # Static assets
│   │   └── README.md                    # Documentación principal
├── nerdAfore/                           # Carpeta que enlaza con el bot llamado NerdAfore
├── .gitignore                           # Docker Compose config
├── README.md                            # Project documentation
├── CONTRIBUTORS.md                      # Contributing guidelines
```
